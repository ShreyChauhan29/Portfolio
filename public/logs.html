Important — how this works: GitHub Pages is a static host with no server, so this page shows the activity recorded in the current browser only. It is perfect for verifying that the protection and logging work, but it cannot see other people's devices.

To see every visitor (count, country, pages, time on site, and the blocked-action events) in one private dashboard: create a free account at goatcounter.com, then put your site code into site-guard.js (the GOATCOUNTER_CODE value at the top) and republish. Every page already sends visits and protection events there once the code is set. Your dashboard stays private to your GoatCounter login — nobody else can open it.
